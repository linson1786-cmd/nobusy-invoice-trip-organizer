## 使用人指引

### 首次使用

1. 复制 `交付包/V1.0/` 到使用人电脑
2. 安装依赖：`pip install pdfplumber pytesseract Pillow pandas openpyxl`
3. 运行初始化：V1.0 历史包使用 `cd scripts && python3 setup.py init`；V1.0.8 起源码目录使用 `cd scripts/invoice-trip-organizer && python3 setup.py init`
4. 配置邮箱（如需邮件下载功能）
5. 开始使用

### 更新指引

使用人运行任意脚本时，版本管理器会自动检测更新。如需手动更新：

```bash
cd scripts/invoice-trip-organizer
python3 version_manager.py
```
